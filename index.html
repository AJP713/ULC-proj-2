Text

  • Select only 1 or 2 fonts.
  • Sans-serifs are always a great choice.
  • Limit the use of heavily stylized fonts.
  • Chose a large readable font-size.
  • Font size and weight should correspond with the importance of the element.

Color

  • Limit the color pallet to 3 or 4.
  • Have high contrast between the text and backgound.
  • Use the contrast of light and dark colors to guide the user's focus.
  • Do not use clashing or dstracting color combinations.

Image

  • Choose high quality, clear images
  • Do not distort images by changing the ratio of the length/height.
  • Be consistent with the styles of pictures, illustrations and graphics.
  • Choose images that will focus the user attention not distract it.

Layout

  • Use whitespace to make important elements stand out.
  • Do not clutter the screen with too much at once.
  • Create clear distinctions between separate sections.
  • Balance the size of different elements.

Alignment

  • Be consistent with the alignment of text, images and borders.
  • Use margins and padding to create white space.
  • Use grid layout to organize content in rows and columns.

Responsiveness

  • Be aware of how the page will look at different screen/window sizes.
  • Use units like %, vw and vh over pixels when possible to allow the elements to change size.
  • Design different grid layouts for different screen sizes.
